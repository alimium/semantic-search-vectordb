BENEFIT-COST ANALYSIS TOOLKIT V5.1 SYSTEM REQUIREMENTS 
You need to have full local administrative rights on the computer on which you will install the 
software, or have someone who has adequate rights perform the installation for you. 
Additionally, manually perform a Microsoft (MS) Windows update to ensure that you have latest 
service packs and security patches prior to performing the installation. 
 
 
 
 
 
 
The following are the minimum system requirements to run the BCA software successfully: 
 
 
 
 
Processor 
 
32-Bit Systems: 1-Gigahertz (GHz) or faster processor recommended. 
64-Bit Systems: 2-GHz or faster processor recommended. 
 
 
 
 
 
 
Framework 
 
If the following items are not already installed, the BCA installation program will attempt to install 
them. 
 
 
 
 
Microsoft .NET Framework 2.0 and 3.5 
SQL Server Express 2008 
 
 
Operating Systems 
 
Windows XP with Service Pack 2 or later 
Vista (32-bit or 64-bit) with Service Pack 2 or later 
Windows 7 (32-bit or 64-bit) with Service Pack 1 or later 
Windows 8 with Service Pack 1 or later 
Windows 8.1 
 
 
 
 
 
 
 
Memory 
 
1 gigabyte (GB) minimum; 2 or more GBs recommended. 
 
 
Hard Disk 
 
Approximately 2 GB of available hard-disk space for the recommended installation. 
 
Display 
 
Super VGA (1,024x768 pixels) or higher-resolution video adapter and monitor. 
 
Other Requirements 
 
 
 
1. Microsoft Internet Explorer 6.0 Service Pack 1 or later. 
2. PDF reader, such as Adobe Reader 7.0 or later. 
3. An unzip utility.
