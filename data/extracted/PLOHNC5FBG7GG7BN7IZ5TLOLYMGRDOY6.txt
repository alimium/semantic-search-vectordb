 
 
VennDiagram Documentation 
 
Description:    
 
Displays a Venn diagram. 
Author:  
 
 
Joshua Gould (Broad Institute), gp-help@broad.mit.edu 
Summary:    
 
 
Displays a Venn diagram illustrating set overlap. See the file format section of the 
GenePattern documentation for a description of the grp file format. 
 
 
Parameters 
feature.list.one 
Feature list one - .txt, .grp 
feature.list.two 
Feature list two -  .txt, .grp 
feature.list.three 
Feature list three - .txt, .grp
